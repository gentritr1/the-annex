# Content authoring

## A complete case needs

1. One question the player performs through actions.
2. One convergence point that can support at least three defensible decisions.
3. Two or more field sites with mutually exclusive methods.
4. Evidence whose source, status, and contradiction are explicit.
5. At least one reconstruction with more than one valid interpretation.
6. Persona effects tied to method tags, not just the ending.
7. A debrief that reports consequences without assigning a morality score.

## Authoring rules

- Write actions as verbs: “Authenticate the custody chain,” not “Procedure route.”
- State the likely cost or risk before commitment when the character would know it.
- A locked option must say how it unlocks.
- Never hide canonical facts in generated prose.
- Reuse method tags only when their systemic meaning is stable.
- Keep one event-log sentence for every state change so playtests can reconstruct cause.

## Safe extension sequence

Start by duplicating the current case content into a new module. Let two cases stabilize before extracting a generalized campaign layer. This keeps the domain explicit while the real reuse pattern is still emerging.
